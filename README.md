# 🔒 SecuritySafe

Desktopowa aplikacja, służąca do bezpiecznego przechowywania haseł, notatek oraz poufnych danych. Projekt stawia na intuicyjny interfejs graficzny i prywatność użytkownika.

## ✨ Funkcjonalności
- **Menedżer Haseł:** Bezpieczne przechowywanie danych logowania do różnych serwisów.
- **Poufne Notatki:** Miejsce na zapisywanie ważnych informacji, do których dostęp ma tylko właściciel.
- **Interfejs GUI:** Nowoczesny i responsywny wygląd zbudowany przy użyciu JavaFX i plików FXML.
- **Zarządzanie sesją:** System logowania chroniący dostęp do bazy danych aplikacji.
---
## 📁 Architektura projektu
- **Model-View-Controller (MVC):** Wyraźne oddzielenie logiki biznesowej od warstwy prezentacji (FXML).
- **Zasoby:** Wykorzystanie stylów CSS do personalizacji wyglądu aplikacji.
- **Maven:** Zarządzanie zależnościami i cyklem życia projektu.
